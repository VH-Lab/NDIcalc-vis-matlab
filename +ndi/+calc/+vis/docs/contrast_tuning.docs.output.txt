This calculation object produces a document of type 'ndi_calculation_contrasttuning'.

Superclasses: ndi_document, ndi_document_app, contrast_tuning

The primary purpose of the ndi_calculation_contrasttuning document is to hold the results of a contrast_tuning fit. The contrast tuning fit document has the following fields:

properties: 
			response_units:                         The units of response		
			response_type:                          The category of the response (e.g. mean)           
		
tuning_curve: 
			contrast:                               The set of contrast values 			
			mean:                                   The mean response 			
			stddev:                                 The standard deviation on the set of generated responses
			stderr:                                 The standard error on the set of generated responses		
			individual:                             The individual generated response values
			control_stddev:                         The standard deviation on the set of generated response values of the control group			
			control_stderr:                         The standard error on the set of generated response values of the control group						
		
significance: 
			visual_response_anova_p:                The p-value relating the different mean response values between the raw and control groups
			across_stimuli_anova_p:                 The p-value relating the different mean response values generated by different stimuli
		
fitless: 
			interpolated_c50:                       The contrast that interpolates the 1/2 maximum response value
                                            
		
fit: 
			naka_rushton_RB_parameters:             The 12 fit parameters (defined in the general document) used in the Naka-Rushton function to fit a contrast tuning curve 
			naka_rushton_RB_contrast:               The range of contrast values 	
			naka_rushton_RB_values:                 The range of response values 
			naka_rushton_RB_pref:                   
			naka_rushton_RB_empirical_c50:          The empirical fit contrast that gives 50% of response to 100% contrast (max differs from C50 of the fit because max response is not usually at 100% contrast) 
			naka_rushton_RB_r2:                     The fit r^2 value 
			naka_rushton_RB_relative_max_gain:      The relative max gain     
			naka_rushton_RB_saturation_index:       The saturation index         
			naka_rushton_RB_sensitivity:			The 1x10 vector of contrast sensitivity 
			naka_rushton_RBN_parameters:            The 12 fit parameters (defined in the general document) used in the Naka-Rushton function to fit a contrast tuning curve         
			naka_rushton_RBN_contrast:              The range of contrast values     
			naka_rushton_RBN_values:                The range of response values 
			naka_rushton_RBN_pref:                  
			naka_rushton_RBN_empirical_c50:			The empirical fit contrast that gives 50% of response to 100% contrast (max differs from C50 of the fit because max response is not usually at 100% contrast) 
			naka_rushton_RBN_r2:					The fit r^2 value 
			naka_rushton_RBN_relative_max_gain:		The relative max gain  
			naka_rushton_RBN_saturation_index:		The saturation index         	
			naka_rushton_RBN_sensitivity:			The 1x10 vector of contrast sensitivity 
			naka_rushton_RBNS_parameters:			The 12 fit parameters (defined in the general document) used in the Naka-Rushton function to fit a contrast tuning curve 	
			naka_rushton_RBNS_contrast:             The range of contrast values   
			naka_rushton_RBNS_values:               The range of response values 
			naka_rushton_RBNS_pref:                 
			naka_rushton_RBNS_empirical_c50:        The empirical fit contrast that gives 50% of response to 100% contrast (max differs from C50 of the fit because max response is not usually at 100% contrast)     	
			naka_rushton_RBNS_r2:                   The fit r^2 value 
			naka_rushton_RBNS_relative_max_gain:	The relative max gain  		
			naka_rushton_RBNS_saturation_index:		The saturation index       	
			naka_rushton_RBNS_sensitivity:			The 1x10 vector of contrast sensitivity
